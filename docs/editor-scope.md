# 编辑器范围（最小可用版本）

最小可用版本只支持新手题目所需的 Vim 子集。

## 模式
- 普通模式
- 插入模式

## 移动
- `h`, `j`, `k`, `l`
- `w`, `b`, `e`
- `0`, `$`, `^`
- `gg`, `G`
- `{`, `}`（段落跳转）

## 编辑
- `i`, `a`, `o`, `I`, `A`, `O`
- `x`, `X`
- `dd`, `yy`, `p`, `P`
- `u`, `Ctrl-r`
- `r`（替换单字符）

## 文本对象
- `ciw`, `diw`, `yaw`
- `ci"`, `ci'`, `ci(`, `ci{`, `ci[`（基础成对对象）

## 搜索与替换
- `/`, `?`, `n`, `N`
- `:s`（当前行）
- `:%s`（全文件）

## 重复
- `.`
- 计数前缀，如 `3w`、`2dd`

说明：
- 本范围是对内容作者的约束。
- 高级功能（可视模式、寄存器、宏等）默认不在最小可用版本范围内。
